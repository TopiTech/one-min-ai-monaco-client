サーバー設定とAPI情報

            

              このアプリケーションはブラウザからローカルの中継Expressサーバーにアクセスし、ローカルサーバーが安全に
              1min.ai API と通信を行います。APIキーがフロントエンドに露出することはありません。
            

            
    
              
  1. 
                プロジェクトルートの .env.example をコピーして
                .env を作成します。
              
    2. 
              
  2. ONE_MIN_AI_API_KEY にあなたの 1min.ai API キーを設定します。
    3. 
              
  3. 
                ターミナルで npm install を実行して依存関係をインストールします。
              
    4. 
              
  4. npm start を実行してサーバーを起動します。
    5. 
            

            

              

                環境変数デフォルト値
              

              

                DEFAULT_CHAT_MODEL: gpt-4o-mini

                DEFAULT_IMAGE_MODEL: gpt-image-2

                DEFAULT_IMAGE_EDITOR_MODEL: gpt-image-2
