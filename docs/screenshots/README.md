# Screenshots

スクリーンショットを配置するディレクトリです。
GitHubリポジトリのREADMEで参照する画像をここに保存します。

## 推奨スクリーンショット一覧

| ファイル名             | 内容                             |
| ---------------------- | -------------------------------- |
| `chat.png`             | 通常チャット画面                 |
| `image-generation.png` | 画像生成 / テキスト編集画面      |
| `coding-editor.png`    | コーディングエディタ画面         |
| `agent-timeline.png`   | AIエージェントのタイムライン表示 |
| `inline-chat.png`      | インラインチャット機能           |

> 実際のスクリーンショットはご自身の環境で撮影し、配置してください。
